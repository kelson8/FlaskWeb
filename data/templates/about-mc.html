Some mods in the pack


I have Mekanism, Thermal Expansion, Enderio and a ton of other cool mods. The links to the mods and the mod list will be in the wiki once I get it setup.

Technical details below

The forge server uses computer craft and I have been learning how to interface with the computers in the pack. I use KubeJS for stuff like making new recipes, and quite a bit of scripting features for the server so I can customize the KelsonCraft server to my liking and try to add what is requested.
